Automobile Hood Latch Modelling

  • Animated and designed working model of a hood latch for an automobile
  • Used a quarter-car model with State-Space analysis to retrieve the expected forces on the latch when going over speedbumps
  • Conducted fatigue analysis, frequency analysis, and all other forces on the hood latch to ensure it does not fail.

Softwares used:

  • SOLIDWORKS
  • MATLAB

Digital and Analog Filter Designer

  • Produced a Fast Fourier Transform function using the recursive Cooley-Tukey algorithm to process larger data sets to achieve the desired O(nlogn) time complexity using a radix-2
  • Implemented a Windowed-Sinc filter to achieve the desired frequency bands with exceptional roll-off
  • Allowed users to apply their desired cutoff frequencies and digital bandwidth with numerous types of windows (Blackman Harris, Rectangular, Hamming, Kaiser)

Software Used:

  • MATLAB
  • MATLAB App Designer

Active Suspension Tuned on a Half-Car Model

  • Tested and tuned active suspension with a PID and a fuzzy logic controller on ISO8608 Road Profile
  • Tuned a PID controller using the Ziegler-Nichols method to minimize displacement and ensure high rider comfort
  • Created a Simulink model to determine the resulting outputs and displacements from the two controllers

Speed Control of a BLDC Computer Fan

  • Constructed a circuit with a BJT, 7-segment displays, and 8051 microcontroller to control a BLDC computer motor with a modifiable PWM signal
  • Programmed in C to develop the PWM signal using the integrated timers found in the 8051, to ensure the best performance
  • Calculated the current RPM of the fan using the hall sensor, and the second timer on the 8051, displaying it on the four 7-segmant displays used

Software and Tools Used:

  • Embedded C
  • Proteus
